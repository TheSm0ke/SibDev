Для запуска нужен python3 и django 3xx.
Проект имеет 1 приложение History, в котором имеются htmk шаблоны используешься для рендеринга страниц.
Создан 1 класс UploadDocumentForm, для работы с формой и имеет 1 поля для файла.
В моделях используется SavedForm для сохранения информацию в бд, и имеет 3 поля username(Имя пользователя, который совершал покупку), spent_money(Количетво потраченных денег), gems(Использует камни).
При неудачных загрузках файла, будет выведена ошибка. Так же при удачных обработках файла выведится последняя успешная загруженная строка, можно посмотреть все строки нажав на кнопку All objects.
Что бы выйти из всех объектов достаточно нажать на SibDev.
